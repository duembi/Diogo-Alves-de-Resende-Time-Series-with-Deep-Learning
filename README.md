# Diogo-Alves-de-Resende-Time-Series-with-Deep-Learning
📊 Time Series Forecasting with Deep Learning

Bu repository, Diogo Alves de Resende’nin Udemy’deki Time Series Forecasting kursu kapsamında oluşturduğu kodlar ve örneklerden oluşmaktadır.

⚠️ Not: Buradaki kodlar bana ait değildir; eğitim amaçlı olarak kurs materyallerini kendi GitHub hesabımda saklamak için eklenmiştir.

🚀 Amaç
Time series (zaman serisi) kavramlarını öğrenmek ve uygulamak
Farklı forecasting yöntemlerini incelemek
Deep learning modellerini görmek ve çalıştırmak
📂 İçerik
Zaman serisi temelleri
ARIMA, SARIMA gibi istatistiksel modeller
LSTM, GRU ve Transformer tabanlı modeller
🛠️ Kullanılan Teknolojiler
Python, Pandas, NumPy
Scikit-learn
TensorFlow / PyTorch
Statsmodels
